# 🎨 AI Image Enhancer - Remini Style by Naim Xbz

A powerful, browser-based AI image enhancement tool built with pure HTML, CSS, and JavaScript. No server required - all processing happens locally in your browser!

## ✨ Features

- 🚀 **4K Upscaling** - Enhance images up to 4x resolution
- 🎭 **AI-Powered Enhancement** - Intelligent detail restoration
- 📷 **Camera Support** - Take photos directly from your camera
- 🎨 **Multiple Filters** - Vivid, Warm, Cool, B&W, Sepia, Dramatic
- ⚡ **Real-time Preview** - See changes instantly
- 📊 **Comparison Slider** - Compare before/after easily
- 🔧 **Advanced Controls**:
  - Brightness, Contrast, Saturation
  - Sharpness & Denoise
  - Smart Presets (Auto, Portrait, Landscape, Low Light)
- 📱 **Fully Responsive** - Works on all devices
- 🔒 **Privacy First** - All processing done locally
- ⌨️ **Keyboard Shortcuts** - Quick actions
